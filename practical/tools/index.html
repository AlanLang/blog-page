个人工具箱（2023 年 1 月）

2023-01-12

看了 Yu Zhang 大佬的个人工具箱（2022 年 5 月）的博文，正巧自己也是一个工具控，所以仿照大佬的格式，也整理了下自己目前所用的所有工具，也希望能对其他人有所参考。

下文所提及的所有软件服务均为本人购买/订阅或学生资格所享有的教育授权，请支持正版！

主要设备

💻 电脑及配件

  • Apple iMac (2021) M1, 16GB RAM, 500GB SSD
    家里的主力电脑，在家办公学习时的生产力工具，就是屏幕小了一点，应该再配个拓展屏
  • OpenCore Hackintosh，3.6 GHz 八核 Intel Core i7，16GB RAM，500GB SSD
    公司里的主力电脑，性能依旧能打
  • Apple MacBook Pro (16-inch，2016) 银色，2.4 GHz 八核 Intel Core i7，16GB RAM，500GB SSD
    刚毕业时买的笔记本，现在专门用在开会或出差时使用
  • 戴尔（DELL）P2715Q 4K 显示器
    公司台式机的显示器，用了好几年了，显示质量优异，性价比也还可以
  • Rost 60 客制化键盘，蒂芙尼线性轴，文若素白键帽 最满意的一款客制化键盘，美观简约，配合自己刷的 qmk 固件能很大得提高生产力
  • Apple Magic Keyboard 键盘，白色 买 iMac 时带的，主要用上面的指纹识别解锁电脑
  • Glorious Model O 鼠标 主力鼠标，个人认为最好看的洞洞鼠，颜控福利
  • Apple Magic Mouse 2 鼠标 买 iMac 时带的，没怎么用过
  • Apple Magic Trackpad 触控板，白色 苹果的触摸板一直很好用，平时会配合鼠标使用，用在一些需要滑动的场景下

📱 手机及平板

  • Apple iPhone 13 mini 蓝色，128GB 个人作为小屏爱好者，这款手机符合我对手机所有的需求，是我最满意的苹果设备，不过从 14 开始苹果就砍掉了 mini 系列，后面可能买不到这么小巧的手机了
  • Apple iPad Mini6 紫色，256GB 在家里时基本不会使用手机，而是使用 ipad 作为替代，主要用来观看文章和视频，还有刷推

🎮 休闲娱乐

  • 索尼 65X9500H HDR 体验最好的电视，无广告，和电视盒子配合良好，画质和音质体验都很不错
  • NVIDIA SHIELD TV Pro 地表最强的 Android 电视盒子
  • Apple TV 4K 为了和家里的其他苹果设备紧密配合，最终还是换成了 Apple TV，用起来感觉比 SHIELD TV 流畅一点，整个界面及 APP 的 UI 也更加统一和谐
  • Nintendo Switch 主机 之前一直用来玩动物森友会，目前已经吃灰很久了
  • Nintendo Switch Pro 手柄 手感比 Joy-Con 好太多，主机模式下一般不会用 Joy-Con
  • XBox Series X 一口气购买了 3 年的 Xbox Game Pass，一大堆游戏随便玩，微软大气。
  • Apple HomePod mini 音箱 * 2 一开始作为家里电视的拓展音箱，但是可能因为个人耳朵的问题听不出音质的区别，所以现在拿到每个房间专门用来听歌了
  • 小爱智能音箱 联动家里的各类智能家居

🎧 可穿戴设备

  • Apple Airpods Pro 降噪耳机 通勤必备，降噪体验不错，能够隔绝地铁上大部分噪音，不过用了两年了电池已经不行了
  • Apple Watch Series4 为了记录健康数据并提醒我锻炼身体，然而并没有锻炼身体，目前用的最多的就是作为地铁卡使用，不用掏手机

📦 网络及存储

  • MikroTik RB450Gx4 功能强大的 Ros 软路由，比 OpenWrt 稳定，目前作为家里的主路由
  • OpenWrt 软路由 作为家里的旁路由使用，主要负责广告过滤和科学上网等
  • GLiNet MT1300 便携路由器，OpenWrt 系统，出差时使用
  • Deco X68 * 2 作为每个房间里的 AP，支持 k/v/r 无线漫游。
  • Synology DS1821+ 作为整个家庭的存储服务器
  • unraid 主机 作为整个家庭的应用服务器

🎥 拍照摄影

  • DJI Mini 2 作为一款入门级无人机，一直还没有入门
  • DJI OM 3 头脑一热就买了，不太好用

自建服务

  • Plex Media Server 自建的流媒体服务，用于在手机平板和电视上观看剧集和电影
  • Synology Drive 群晖自带的文件同步服务，类似于坚果云
  • Synology Photos 群晖自带的图片同步服务
  • fava Beancount 的 web 页面
  • Nginx Proxy Manager 反向代理管理系统，它基于 NGINX，具有漂亮干净的 Web UI。还可以获得受信任的 SSL 证书，并通过单独的配置、自定义和入侵保护来管理多个代理
  • movie-rebot 配合 qBittorrent 和 Plex 实现家庭影音自动化，会自动监听我的豆瓣想看，并将媒体下载到 Plex 中
  • Grafana 负责搭建一些常用的看板
  • vaultwarden 密码管理工具服务端
  • youtube-dl-server 用于下载 youtube、B 站 等媒体服务的视频
  • Miniflux 用来管理 RSS 订阅
  • RSSHub 一个开源、简单易用、易于扩展的 RSS 生成器，可以给任何奇奇怪怪的内容生成 RSS 订阅源
  • qBittorrent 一款开源免费的种子和磁力链接下载工具

电脑软件

🛠 系统拓展

  • AppCleaner 轻量级的 Mac 软件卸载工具
  • LaunchBar 一直在用的一款应用程序启动器
  • QSpace 是一款简洁高效的多窗格文件管理器，并可扩展为 FTP、SFTP、WebDAV、OneDrive 客户端。
  • Fig 惊艳的终端自动补全工具，整合苹果原生 Terminal、iTerm2、VSCode 里的终端，支持 Git、文件夹、NPM、Kubernetes、Docker、AWS 等自动补全
  • Rectangle 窗口管理工具，可以让用户使用键盘快捷键移动和调整 MacOS 中的窗口大小
  • iStat Menus 系统状态显示，主要显示 CPU、内存和网络等状态
  • CleanShot X 超级好用的截图工具，不止可以截图，还可以录屏
  • AltTable 窗口切换管理工具

📝 笔记工具

  • Logseq 主力的生活记录、任务管理和笔记工具。基于双向链接打造了自己的信息管理系统，而且通过 Journal 以文本条目的形式记录没有心智负担
  • Notion 主要作为个人数据库，作为软件订阅管理等数据类的记录
  • MWeb Pro 用来写文章草稿，写完后会发布到博客等其他平台
  • Typora Markdown 编辑器，买了 MWeb Pro 后没怎么用过了

🧰 实用工具

  • PicGo 图床管理工具，主要写博客的时候上传图片到图床并且生成自定义的 Markdown 格式，可以直接嵌入文章
  • BetterZip 压缩/解压工具
  • Rime 鼠须管 输入法
  • Synology Drive 群晖官方的文件同步工具，类似于坚果云，数据会在自己家的群晖上

👨🏻‍💻 开发工具

  • 编辑器
    • VS Code 使用频率最高的代码编辑工具
    • neovim 优雅的 vim 编辑器
    • Xcode 用来开发 MacOS 和 iOS
  • 终端
    • iTerm2 主力终端工具
  • 服务器管理
    • ssh 配合 Warp 和 iTerm2 的 Profile 来连接服务器用
    • Transmit 用来和服务器之间拷贝文件
  • 数据库管理
    • SQLPro Studio 支持多种数据库的数据库管理工具
  • 接口调试
    • Postman 一直在用的接口调试工具

手机 App

🛠 系统拓展

  • 熊猫吃短信 好用的短信过滤工具
  • Due 强大的定时提醒工具
  • Streaks 习惯养成工具
  • Scriptable 可以自己制作一些好看的桌面小组件
  • Bark 消息推送服务

🔐 安全工具

  • Bitwarden 密码管理工具
  • Authy 双重验证工具

🌏 网络工具

  • Surge 代理工具
  • Thor 抓包工具
  • SpeedTest 网络测速工具

📝 记录工具

  • Notion Notion IOS 客户端
  • flomo 记录闪念
  • Things 记录代办，只用在身边没有电脑的时候，会同步到 Logseq 中

📨 信息管理

  • Cubox 稍候阅读工具，拥有良好的交互体验和强大的全文搜索功能
  • Pocket 稍候阅读工具，用了 Cubox 后没再用过
  • GitHub GitHub iOS 客户端
  • Twitter 主要国外资讯获取与发布平台
  • Reeder 聚合 RSS 阅读工具
  • ComicGlass PDF 阅读工具
  • 微信读书 看书
  • 知乎 iOS 端
  • 豆瓣 iOS 端

💰 金融理财

  • 支付宝 日常最主要支付工具
  • 天天基金 用来购买基金
  • 东方财富 用来购买股票
  • 招商银行 管理招商银行卡
  • 掌上生活 管理招商银行信用卡
  • 中国农业银行 管理农业银行卡

💬 通讯社交

  • Gmail 查收邮件
  • Telegram 订阅群组或聊天
  • Instagram 发布拍摄的照片
  • 微信 没办法不用，主要和家里人聊天
  • 企业微信 工作需要

🎸 影音娱乐

  • YouTube 主要用的在线视频服务，购买了 Premium，用来看技术视频
  • nPlayer 播放手机本地的视频或者 NAS 里的视频
  • Infuse 用来播放 Plex 里的视频，优势是可以播放蓝光原盘
  • Plex 订阅了 Infuse 后没怎么用过了
  • Spotify 主要的音乐播放器
  • Game Pass 用来给 Xbox 下载游戏
  • 哔哩哔哩 观看番剧
  • 小宇宙 听播客

👨‍⚕️ 健康管理

  • Workout 健身锻炼记录工具
  • 喝水时间 记录喝水的工具，保证每天补充足够的水分

总结

以上就是我在 2023 年 1 月所自用的工具箱，花了好几天，把电脑和手机的 App 都彻底整理分类了一遍，随着使用时间越来越长，一直在做减法，也在不断优化自己的使用，希望能够对大家有所参考。
